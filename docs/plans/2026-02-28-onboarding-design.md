# H-08: オンボーディング体験 - 設計書

## 概要

初回訪問時に82行のフルサンプルではなく、簡潔なウェルカムメッセージをエディタ内容として表示。特殊UIコンポーネント不要。

## 方針

- ウェルカムコンテンツは通常のマークダウンとしてエディタに表示（Engineer推奨）
- onboardingフラグ不要（localStorageにコンテンツ有無で判定）
- フルサンプルはテンプレートメニュー「サンプルコンテンツ」として引き続き利用可能
- プレースホルダーをi18n化

## 変更ファイル

| ファイル | 変更 |
|---------|------|
| `constants/templates/welcomeContent.md` | **新規** ウェルカムメッセージ |
| `MarkdownEditorPage.tsx` | defaultContent → welcomeContent に差し替え、プレースホルダーi18n化 |
| `i18n/en.json`, `i18n/ja.json` | プレースホルダーキー追加 |

## ウェルカムコンテンツ

3-5行の簡潔なガイダンス + 操作ヒント。日英共通マークダウン構文。
