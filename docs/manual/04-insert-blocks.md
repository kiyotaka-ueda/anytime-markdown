# 4. 図表・コードを挿入する

ブロック要素はスラッシュコマンド（`/`）またはショートカットキーで挿入します。各ブロックにはツールバーが表示され、編集・全画面表示・削除などの操作が可能です。

## テーブル

設計書の仕様一覧、API パラメータ表、テーブル定義などに使用します。

### テーブルの挿入

1. `/table` と入力するか `Ctrl+Alt+T` を押します。
2. 3行 x 3列のテーブルが挿入されます。

### セル間の移動

- `Tab`: 次のセルへ移動
- `Shift+Tab`: 前のセルへ移動

### フローティングツールバー

テーブルを選択すると、以下の操作が可能なツールバーが表示されます。

| 操作 | 説明 |
|------|------|
| 列の追加 / 削除 | 現在のセルの右側に列を追加、または現在の列を削除 |
| 行の追加 / 削除 | 現在のセルの下に行を追加、または現在の行を削除 |
| セル配置 | 左揃え / 中央揃え / 右揃え |
| 行の移動 | 上下に行を入れ替え |
| 列の移動 | 左右に列を入れ替え |
| 全画面編集 | テーブルを全画面で編集（検索・置換機能付き） |

### 全画面テーブル編集

テーブルツールバーの **全画面** ボタンをクリックすると、テーブル専用の編集画面が開きます。

全画面編集では **検索・置換** 機能が利用できます（大文字小文字区別、単語単位、正規表現に対応）。大量のデータを含むテーブルの一括修正に便利です。

## Mermaid ダイアグラム

設計書のアーキテクチャ図、シーケンス図、ER図などを Mermaid 記法で作成できます。

### ダイアグラムの挿入

1. `/mermaid` と入力するか、ツールバーのダイアグラムボタンから **Mermaid** を選択します。
2. 空の Mermaid ブロックが挿入されます。
3. ブロックのツールバーから **全画面** ボタンをクリックして編集画面を開きます。

### 全画面編集画面

| 領域 | 説明 |
|------|------|
| 左: コードエディタ | Mermaid コードを記述。行番号表示、Tab インデント対応 |
| 右: プレビュー | SVG でリアルタイムレンダリング。ズーム・パン操作可能 |
| Code / Config タブ | Code: ダイアグラム本体。Config: `%%{init:...}%%` の設定を分離編集 |

### サンプルダイアグラム

**サンプル** ボタンから 23 種類のサンプルを挿入できます。

| 設計書での用途 | 推奨ダイアグラム |
|---------------|----------------|
| システム構成図 | Flowchart, Architecture, Block |
| シーケンス図 | Sequence |
| クラス図 | Class |
| ER 図 | ER |
| 状態遷移図 | State |
| マインドマップ（要件整理） | Mindmap |
| プロジェクト計画 | Gantt, Timeline |
| C4 モデル | C4 |

### ダイアグラムのエクスポート

ツールバーの **キャプチャ** ボタンで、レンダリングされたダイアグラムを PNG 画像として保存できます。他のドキュメントへの貼り付けに利用します。

## PlantUML ダイアグラム

Mermaid で表現しにくいダイアグラム（詳細なクラス図、コンポーネント図等）には PlantUML を使用します。

### ダイアグラムの挿入

1. `/plantuml` と入力するか、ツールバーのダイアグラムボタンから **PlantUML** を選択します。
2. 初回使用時、ダイアグラムコードを外部サーバー（plantuml.com）に送信する旨の同意ダイアログが表示されます。

### サンプルダイアグラム

12 種類のサンプルが用意されています。

| 設計書での用途 | 推奨ダイアグラム |
|---------------|----------------|
| API シーケンス図 | Sequence |
| クラス設計 | Class |
| ユースケース図 | UseCase |
| デプロイ構成図 | Deployment, Component |
| データモデル | ER |
| 業務フロー | Activity, Swimlane |

### 全画面編集

Mermaid と同様に、Code / Config タブ分離、ライブプレビュー、ズーム・パン、キャプチャ機能が利用できます。Config タブでは `skinparam` や `!theme` を分離して編集できます。

## コードブロック

設計書にソースコードの例、設定ファイル、SQL クエリなどを挿入します。

### コードブロックの挿入

1. `/codeBlock` と入力するか `Ctrl+Alt+C` を押します。
2. コードブロックが挿入されます。
3. ツールバーの言語ラベルをクリックして、言語を指定します（自動検出もされます）。

### 対応言語

TypeScript, JavaScript, Python, Java, Go, Rust, C/C++, C#, SQL, Bash, HTML, CSS, JSON, YAML, XML, Ruby, PHP, Swift, Kotlin, Markdown, Lua, R, Perl など約37言語に対応しています。

### 全画面編集

コードブロックのツールバーから **全画面** ボタンをクリックすると、シンタックスハイライト付きの全画面エディタが開きます。

### 設計書での活用例

- API リクエスト/レスポンスの JSON サンプル
- SQL のテーブル定義（DDL）
- 設定ファイル（YAML / JSON）
- 擬似コードやアルゴリズムの説明

## 数式（KaTeX）

技術仕様書やアルゴリズム設計書で数式を使用する場合に利用します。

### 数式の挿入

1. `/math` と入力します。
2. 数式ブロックが挿入されます。
3. KaTeX 記法で数式を記述します。

### サンプル数式

7 種類のサンプル（分数、総和、積分、行列、極限、偏微分、場合分け）が用意されています。

## 画像

設計書にスクリーンショット、ワイヤーフレーム、外部の図を挿入します。

### 画像の挿入方法

| 方法 | 操作 |
|------|------|
| URL 指定 | `Ctrl+Alt+I` で URL を入力 |
| ドラッグ&ドロップ | 画像ファイルをエディタにドラッグ |
| クリップボード | スクリーンショットを撮影後、`Ctrl+V` で貼り付け |

### 画像の編集

画像ブロックのツールバーから以下を編集できます。

- **Alt テキスト**: 画像の代替テキスト（アクセシビリティ用）
- **Title**: ホバー時に表示されるツールチップテキスト

### 画像のリサイズ

画像の右下に表示されるグリップをドラッグして、表示幅を変更できます。キーボード操作にも対応しています。

## HTML ブロック

Markdown では表現しにくいレイアウト（フォーム、カード UI、グリッドレイアウト等）を HTML で直接記述できます。

### HTML ブロックの挿入

1. `/html` と入力します。
2. 6 種類のサンプル（Form, Table, Card, Navigation List, Details/Summary, CSS Grid Layout）から選択できます。

### 全画面編集

コードエディタとライブプレビューが並列表示されます。HTML は DOMPurify でサニタイズされた上で表示されます。

## ブロック要素の共通操作

すべてのブロック要素に共通する操作です。

| 操作 | 説明 |
|------|------|
| **ドラッグ移動** | ブロック左側のハンドルをドラッグして位置を移動 |
| **リサイズ** | 画像・ダイアグラム・数式は右下グリップでサイズ変更 |
| **ダブルクリック** | ダイアグラム・数式・HTML はダブルクリックで全画面編集を開く |
| **折りたたみ** | ツールバーの折りたたみボタンで表示を縮小 |
| **コードコピー** | ダイアグラム・数式・HTML のソースコードをクリップボードにコピー |
| **削除** | ツールバーの削除ボタンでブロックを削除 |

## 全画面編集の共通機能

コードブロック、ダイアグラム、数式、HTML の全画面編集画面で共通して利用できる機能です。

| 機能 | 説明 |
|------|------|
| コード / プレビュー分割 | 左右分割表示。ドラッグで幅を調整可能 |
| ライブプレビュー | コード変更がリアルタイムにプレビューに反映 |
| ズーム・パン | ボタンまたはマウスホイールでズーム、ドラッグでパン |
| 行番号表示 | コードエリアに行番号を表示 |
| Tab インデント | Tab キーで2スペースのインデントを挿入 |
| サンプル挿入 | 折りたたみ式のサンプルパネルから選択して挿入 |
