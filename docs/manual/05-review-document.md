# 5. 設計書をレビューする

AI が出力した設計書や、チームメンバーが作成した設計書をレビューする手順を説明します。

## レビューの基本フロー

```
1. 設計書ファイルを開く
2. AI 出力版と比較する（比較モード）
3. 差分を確認し、必要な箇所にコメントを追加（レビューモード）
4. コメントを解決しながら修正
5. 完了後、保存・エクスポート
```

## 比較モードで差分を確認する

AI が出力した設計書と、元の設計書の差分を確認します。

### 比較モードの開始

1. 元の設計書をエディタで開きます（左ペインに表示されます）。
2. ツールバーの **通常 / 比較** 切替ボタンで **比較** を選択します（ショートカット: `Ctrl+Alt+M`）。
3. 画面が左右に分割されます。

![比較モード](images/05-compare-mode.png)

### 比較対象の読み込み

右ペインに AI 出力の設計書を読み込みます。

- **ファイル読み込み**: 右ペイン上部の **ファイル読込** ボタンから .md ファイルを選択
- **直接入力**: 右ペインに直接テキストを貼り付け

### 差分の確認

左右のペインで差異がある箇所がハイライト表示されます。

### ブロック要素の差分

コードブロック・ダイアグラム・数式の全画面編集画面では、**差分表示** タブで行単位の差分を確認できます。

### マージ操作

右ペイン（AI 出力）の内容を左ペイン（元の設計書）に取り込む場合、**マージ** 操作を使用します。マージは右→左方向のみです。

### 比較モードの終了

ツールバーの切替ボタンで **通常** に戻します。左ペインの内容がエディタに残ります。

## レビューモードでコメントを追加する

### レビューモードへの切替

ツールバーのモード切替で **Review** を選択します。

![レビューモード](images/05-review-mode.png)

Review モードでは:
- テキスト編集はできません
- テキスト選択 → コメント追加が可能です
- チェックボックスの操作が可能です
- 図やテーブルはダブルクリックで全画面表示できます

### コメントの追加方法

#### テキストを選択してコメント

1. コメントしたいテキストを選択します。
2. バブルメニューの **コメント** ボタンをクリックするか、`Ctrl+Shift+M` を押します。
3. コメントテキストを入力して確定します。
4. 選択したテキストが黄色の背景でハイライト表示されます。

#### カーソル位置にマーカーコメント

1. テキストを選択せずにカーソルを置きます。
2. `Ctrl+Shift+M` を押すか、`/comment` スラッシュコマンドを使用します。
3. カーソル位置にコメントマーカーが挿入されます。

### 設計書レビューでのコメント例

| 指摘内容 | コメント例 |
|---------|-----------|
| 仕様の不明確さ | 「このAPIのレスポンス形式を明記してください」 |
| セキュリティ懸念 | 「認証トークンの有効期限が未定義です」 |
| 設計上の質問 | 「このテーブルにインデックスは必要ですか？」 |
| 改善提案 | 「キャッシュ戦略の追加を検討してください」 |

## コメントパネルで管理する

### コメントパネルの表示

ツールバーのコメントアイコンをクリックして、コメントパネルを開きます。

### コメントの操作

| 操作 | 説明 |
|------|------|
| **一覧表示** | すべてのコメントが投稿順に表示されます |
| **フィルター** | 「すべて」「未解決」「解決済み」で絞り込み |
| **解決** | コメントを解決済みとしてマーク |
| **解決取消** | 解決済みコメントを未解決に戻す |

### レビュー完了の確認

コメントパネルのフィルターを **未解決** に設定し、残りの指摘事項がないことを確認します。すべてのコメントが解決済みになったら、レビュー完了です。

## Edit モードでコメントに対応する

レビューコメントへの対応は Edit モードで行います。

1. モードを **Edit** に切り替えます。
2. コメントパネルで未解決のコメントを確認します。
3. 該当箇所を修正します。
4. 対応完了後、コメントを **解決** としてマークします。

## レビュー業務の推奨ワークフロー

### AI 出力設計書のレビュー

1. AI に出力させた設計書を `.md` ファイルとして保存
2. 元の要件書または骨格設計書をエディタで開く
3. **比較モード** で AI 出力版を右ペインに読み込む
4. 差分を確認し、採用する箇所をマージ
5. **Review モード** に切り替えてコメントを追加
6. **Edit モード** で修正を実施
7. 全コメント解決を確認して完了

### チームメンバーの設計書レビュー

1. レビュー対象の `.md` ファイルを開く
2. **Review モード** でレビューコメントを追加
3. コメント付きのファイルをメンバーに返却
4. メンバーが修正後、再度レビュー（未解決コメントを確認）
