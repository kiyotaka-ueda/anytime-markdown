# 付録B: スラッシュコマンド一覧

エディタで `/` を入力すると、コマンドメニューが表示されます。コマンド名を入力して絞り込み、Enter または項目クリックで挿入します。

## 見出し

| コマンド | 挿入される要素 |
|---------|--------------|
| `/h1` | 見出し H1 |
| `/h2` | 見出し H2 |
| `/h3` | 見出し H3 |

## リスト

| コマンド | 挿入される要素 |
|---------|--------------|
| `/bulletList` | 箇条書きリスト |
| `/orderedList` | 番号付きリスト |
| `/taskList` | タスクリスト（チェックボックス付き） |

## 引用・注意喚起

| コマンド | 挿入される要素 |
|---------|--------------|
| `/blockquote` | 引用ブロック |
| `/note` | NOTE アドモニション（青） |
| `/tip` | TIP アドモニション（緑） |
| `/important` | IMPORTANT アドモニション（紫） |
| `/warning` | WARNING アドモニション（黄） |
| `/caution` | CAUTION アドモニション（赤） |

## テーブル・区切り

| コマンド | 挿入される要素 |
|---------|--------------|
| `/table` | 3x3 テーブル |
| `/horizontalRule` | 水平線（区切り線） |

## コードブロック・ダイアグラム

| コマンド | 挿入される要素 |
|---------|--------------|
| `/codeBlock` | コードブロック |
| `/mermaid` | Mermaid ダイアグラム |
| `/plantuml` | PlantUML ダイアグラム |
| `/math` | 数式ブロック（KaTeX） |
| `/html` | HTML プレビューブロック |

## ドキュメント要素

| コマンド | 挿入される要素 |
|---------|--------------|
| `/toc` | 目次（見出しから自動生成） |
| `/date` | 今日の日付（YYYY-MM-DD 形式） |
| `/footnote` | 脚注参照 |
| `/comment` | コメントマーカー |
