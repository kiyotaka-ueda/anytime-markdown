# 付録A: キーボードショートカット一覧

Mac では `Ctrl` を `Cmd (⌘)` に読み替えてください。

## ファイル操作

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+Alt+N` | 新規作成 |
| `Ctrl+O` | ファイルを開く |
| `Ctrl+S` | 上書き保存 |
| `Ctrl+Shift+S` | 名前を付けて保存 |
| `Ctrl+Alt+U` | アップロード |
| `Ctrl+Alt+E` | ダウンロード |

## 編集操作

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Shift+Z` | やり直し |
| `Ctrl+Shift+K` | 行を削除 |
| `Shift+Enter` | 段落内改行（ハードブレイク） |
| `Ctrl+Shift+C` | コピー |

## テキスト書式

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+B` | 太字 |
| `Ctrl+I` | 斜体 |
| `Ctrl+U` | 下線 |
| `Ctrl+Shift+X` | 取消線 |
| `Ctrl+Shift+H` | ハイライト（蛍光マーカー） |
| `Ctrl+E` | インラインコード |
| `Ctrl+K` | リンク挿入 |

## 見出し・ブロック

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+Alt+1` | 見出し H1 |
| `Ctrl+Alt+2` | 見出し H2 |
| `Ctrl+Alt+3` | 見出し H3 |
| `Ctrl+Alt+4` | 見出し H4 |
| `Ctrl+Alt+5` | 見出し H5 |
| `Ctrl+Shift+7` | 箇条書きリスト |
| `Ctrl+Shift+8` | 番号付きリスト |
| `Ctrl+Shift+9` | タスクリスト |
| `Ctrl+Shift+B` | 引用（Blockquote） |

## ブロック要素の挿入

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+Alt+I` | 画像挿入 |
| `Ctrl+Alt+R` | 水平線 |
| `Ctrl+Alt+T` | テーブル |
| `Ctrl+Alt+C` | コードブロック |
| `Ctrl+Alt+D` | ダイアグラム |
| `Ctrl+Alt+H` | HTML ブロック |
| `Ctrl+Alt+P` | テンプレート |

## 検索・表示

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+F` | 検索 |
| `Ctrl+H` | 検索・置換 |
| `Ctrl+Alt+O` | アウトラインパネル開閉 |
| `Ctrl+Alt+F` | 全折りたたみ / 全展開 |

## モード切替

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+Alt+S` | Source / Edit モード切替 |
| `Ctrl+Alt+M` | 通常 / 比較モード切替 |

## コメント

| ショートカット | 操作 |
|:---:|------|
| `Ctrl+Shift+M` | コメント追加 |

## テーブル内操作

| ショートカット | 操作 |
|:---:|------|
| `Tab` | 次のセルへ移動 |
| `Shift+Tab` | 前のセルへ移動 |
