# anytime-markdown 対応フォーマット比較

更新日: 2026-03-07

[Claude Code Markdown フォーマット仕様](~/.claude/development_guidelines/claude-code-markdown-format-spec.md) の各セクションに対する anytime-markdown エディタの対応状況を比較する。

---

## 仕様セクション別の対応状況

| # | 仕様セクション | エディタ対応 | Markdown 出力 | 制限事項 |
| --- | --- | --- | --- | --- |
| 1 | ATX 見出し | 対応 | `#` ～ `#####` | レベル1-5のみ。レベル6は未対応 |
| 2 | 水平線 | 対応 | `---` | `-` のみ。`*` `_` は非対応 |
| 3 | フェンスコードブロック | 対応 | ` ``` ` | バッククォートのみ。チルダフェンスは非対応 |
| 4 | テーブル | 対応 | `| --- |` 形式 | セパレータは `| --- |` に正規化される |
| 5 | コードスパン | 対応 | `` ` `` ～ | テーブル行内はバッククォート数が最小化される |
| 6 | リスト | 対応 | `- `/ `1.` | 箇条書きは `-` のみ。`+` `*` `1)` は非対応 |
| 7 | タスクリスト | 対応 | `- [ ]` / `- [x]` | ネスト対応 |
| 8 | ブロック引用 | 対応 | `>` | Admonition 拡張で機能追加 |
| 9 | 強調・太字 | 対応 | `*text*` / `**text**` | `*` のみ。`_` `__` は非対応 |
| 10 | 取り消し線 | 対応 | `~~text~~` | |
| 11 | リンク・画像 | 対応 | `[text](url)` / `![alt](url)` | インラインリンクのみ。参照リンクは非対応。画像は Base64 埋め込み対応 |
| 12 | バックスラッシュエスケープ | 対応（GFM 準拠） | | テーブルセル内は後処理でエスケープ復元 |
| 13 | エンティティ参照 | 対応（GFM 準拠） | | テーブルセル内は後処理でデコード |
| 14 | 改行 | 対応 | 行末 `\` | ハードブレークはバックスラッシュ方式で出力 |

## GFM 仕様外の独自拡張

| 構文 | エディタ対応 | Markdown 出力 | 備考 |
| --- | --- | --- | --- |
| ハイライト | 対応 | `<mark>text</mark>` | HTML タグで出力 |
| 下線 | 対応 | `<u>text</u>` | HTML タグで出力 |
| 数式（インライン） | 対応 | `$expr$` | MathInline 拡張 |
| 数式（ブロック） | 対応 | `$$\n...\n$$` | コードブロック `math` から変換 |
| Mermaid 図 | 対応 | ` ```mermaid ` | コードブロック言語指定 |
| PlantUML 図 | 対応 | ` ```plantuml ` | コードブロック言語指定 |
| 折りたたみ | 対応 | `<details><summary>` | HTML タグで出力 |
| 脚注 | 対応 | `[^1]` / `[^1]:` | FootnoteRef 拡張 |
| Admonition | 対応 | `> [!NOTE]` 等 | ブロック引用を拡張 |
| コメント | 対応 | `<!-- comment-start:id -->` | 入力時に `<span>` に変換される |
| 下付き文字 | HTML のみ | `<sub>text</sub>` | 専用拡張なし |
| 上付き文字 | HTML のみ | `<sup>text</sup>` | 専用拡張なし |
