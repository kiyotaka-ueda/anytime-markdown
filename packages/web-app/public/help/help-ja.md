## 基本操作

エディタは2つのモードと各種ファイル操作に対応しています。

![基本操作](/help/basic-ops.png)

ツールバー右側のボタングループでモードを切り替えられます。

| アイコン | モード | 説明 |
| :------: | ------ | ---- |
| ![WYSIWYG](/help/icons/wysiwyg.png) | WYSIWYGモード | リッチテキスト書式でリアルタイムプレビューしながら編集できます。 |
| ![ソース](/help/icons/source.png) | ソースモード | Markdownを直接編集できます。画像・水平線・テーブル・コードブロック・図の挿入ボタンも使用でき、対応するMarkdown構文がテキスト末尾に挿入されます。 |

ツールバー右側のボタングループで通常モードと比較モードを切り替えられます。

| アイコン | モード | 説明 |
| :------: | ------ | ---- |
| ![通常](/help/icons/normal-mode.png) | 通常モード | 通常の編集モードです。 |
| ![比較](/help/icons/compare-mode.png) | 比較モード | エディタが左右2ペインに分割され、差分がハイライト表示されます。<br>セカンダリ（右側）ペインにはテキストの直接入力またはMarkdownファイルのアップロードが可能です。 |

### ファイル操作

ツールバーからファイル操作が可能です。

| アイコン | 操作 | 説明 |
| :------: | ---- | ---- |
| ![新規作成](/help/icons/create-new.png) | 新規作成 | 現在の内容を消去し、新しいドキュメントを作成します。 |
| ![コピー](/help/icons/copy.png) | コピー | ドキュメント全体をクリップボードにコピーします。 |
| ![インポート](/help/icons/upload.png) | インポート | Markdownファイル(.md)を読み込みます。 |
| ![ダウンロード](/help/icons/download.png) | ダウンロード | ドキュメントをMarkdownファイル(.md)としてダウンロードします。 |

| アイコン | 操作 | 説明 |
| :------: | ---- | ---- |
| ![アウトライン](/help/icons/outline.png) | アウトライン表示 | ツールバーからアウトラインパネルの表示/非表示を切り替えます。 |

### その他

- その他メニュー：ツールバー右端の「⋮」ボタンからヘルプページ、エディタ設定、バージョン情報にアクセスできます。
- 自動保存：コンテンツはブラウザのローカルストレージに自動保存されます。

## テキスト装飾

テキストを選択するとバブルメニューが表示され、書式を適用できます。キーボードショートカットでも操作可能です。

| アイコン | 装飾 | ショートカット | 説明 |
| :------: | ---- | -------------- | ---- |
| ![太字](/help/icons/bold.png) | 太字 | Ctrl+B | 選択テキストを**太字**にします。 |
| ![斜体](/help/icons/italic.png) | 斜体 | Ctrl+I | 選択テキストを*斜体*にします。 |
| ![下線](/help/icons/underline.png) | 下線 | Ctrl+U | 選択テキストに下線を付けます。 |
| ![取り消し線](/help/icons/strikethrough.png) | 取り消し線 | Ctrl+Shift+X | 選択テキストに~~取り消し線~~を付けます。 |
| ![ハイライト](/help/icons/highlight.png) | ハイライト | Ctrl+Shift+H | 選択テキストをハイライト表示します。 |
| ![インラインコード](/help/icons/inline-code.png) | インラインコード | Ctrl+E | 選択テキストを`コード`として表示します。 |
| ![リンク](/help/icons/link.png) | リンク | Ctrl+K | 選択テキストにリンクを設定します。 |

### 図（Mermaid）

コードブロック内でMermaid記法を使って図を作成できます。

![図 - Mermaid](/help/mermaid.png)

- 対応図形：フローチャート、シーケンス図、クラス図、ER図、ステートマシン、ガントチャート、マインドマップ、Gitグラフなど。

フローティングツールバー：

| アイコン | 操作 | 説明 |
| :------: | ---- | ---- |
| ![折りたたみ](/help/icons/fold.png) | 折りたたみ/展開 | 図の表示を折りたたみ・展開します。 |
| ![全画面](/help/icons/fullscreen.png) | 全画面 | 図を全画面モードで表示・編集します。 |
| ![コード表示](/help/icons/code-toggle.png) | コード表示/非表示 | コード入力欄の表示を切り替えます。 |
| ![縮小](/help/icons/zoom-out.png) | 縮小 | 図を縮小表示します。 |
| ![拡大](/help/icons/zoom-in.png) | 拡大 | 図を拡大表示します。 |
| ![リセット](/help/icons/zoom-reset.png) | ズームリセット | ズームとパンを初期状態に戻します。 |

### 図（PlantUML）

コードブロック内でPlantUML記法を使って図を作成できます。

![図 - PlantUML](/help/plantuml.png)

- 対応図形：シーケンス図、クラス図、ユースケース図、アクティビティ図、ステート図、コンポーネント図、デプロイメント図、ER図など。
- PlantUMLの図は外部サーバー（plantuml.com）で描画されます。図コードの送信前に同意を求められます。

フローティングツールバーは Mermaid と同じアイコン・操作が使用でき、加えて定義済みサンプルの挿入や矢印・参加者・クラスなどの要素挿入補助ツールが利用できます。

### 画像

さまざまな方法で画像の挿入やMarkdownファイルのインポートができます。

- ツールバーボタンからURLを指定して画像を挿入できます。
- 画像ファイルをエディタに直接ドラッグ＆ドロップできます。
- クリップボードから画像を貼り付けできます（Ctrl+V / Cmd+V）。
- .mdファイルをエディタにドラッグ＆ドロップしてコンテンツをインポートできます。

## 検索・置換

エディタ内のテキストを検索・置換できます。

![検索・置換](/help/search-replace.png)

- ツールバーまたはキーボードショートカット（Ctrl+F）で検索バーを開きます。
- Ctrl+Hで置換バーを開きます。

| アイコン | オプション | 説明 |
| :------: | ---------- | ---- |
| ![大文字小文字](/help/icons/match-case.png) | 大文字/小文字区別 | 大文字と小文字を区別して検索します。 |
| ![単語単位](/help/icons/match-word.png) | 単語単位 | 完全一致する単語のみ検索します。 |
| ![正規表現](/help/icons/regex.png) | 正規表現 | 正規表現パターンで検索します。 |

- 全画面コードエディタやテーブル表示内でも検索が機能します。

## 比較・マージ

2つのドキュメントを並べて比較・マージできます。

- ツールバー右側の通常モード/比較モードのボタングループで切り替えます。
- 比較モードではエディタが左右2ペインに分割され、差分がハイライト表示されます。
- セカンダリ（右側）ペインには、テキストの直接入力またはMarkdownファイルのアップロードが可能です。
- 差分の取り込み（マージ）操作ができます。

## アウトライン

ドキュメント構造を表示し、セクション間をナビゲートできます。

![アウトラインパネル](/help/outline.png)

| アイコン | 操作 | 説明 |
| :------: | ---- | ---- |
| ![アウトライン](/help/icons/outline.png) | アウトライン表示 | ツールバーからアウトラインパネルの表示/非表示を切り替えます。 |

- アウトライン内の見出しをクリックすると、エディタ内のその位置にジャンプします。
- セクションの折りたたみ/展開で、見出し下のコンテンツを折りたたんだり展開できます。
- ドラッグ＆ドロップまたは上下移動ボタンでセクションを並べ替えできます。
- パネル境界をドラッグしてアウトラインパネルの幅を調整できます。

## エディタ設定

エディタの外観と動作をカスタマイズできます。

![エディタ設定](/help/settings.png)

- 調整可能な設定：行間、フォントサイズ、テーブル幅、エディタ最小幅、エディタ背景色。
