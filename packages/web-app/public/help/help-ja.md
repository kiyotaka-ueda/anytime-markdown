## 基本操作

エディタは3つのモードと各種ファイル操作に対応しています。

![基本操作](/help/basic-ops.png)

ツールバー右側のボタングループでモードを切り替えられます。

| アイコン | モード | 説明 |
| :------: | ------ | ---- |
| ![ビュー](/help/icons/view.png) | ビューモード | 閲覧専用モードです。テキスト編集やブロック操作は無効になります。バブルメニューからコメントの追加は可能です。図・テーブル・画像はダブルクリックで全画面表示できます。 |
| ![WYSIWYG](/help/icons/wysiwyg.png) | WYSIWYGモード | リッチテキスト書式でリアルタイムプレビューしながら編集できます。 |
| ![ソース](/help/icons/source.png) | ソースモード | Markdownを直接編集できます。画像・水平線・テーブル・コードブロック・図の挿入ボタンも使用でき、対応するMarkdown構文がテキスト末尾に挿入されます。 |

ツールバー右側のボタングループで通常モードと比較モードを切り替えられます。

| アイコン | モード | 説明 |
| :------: | ------ | ---- |
| ![通常](/help/icons/normal-mode.png) | 通常モード | 通常の編集モードです。 |
| ![比較](/help/icons/compare-mode.png) | 比較モード | エディタが左右2ペインに分割され、差分がハイライト表示されます。<br>セカンダリ（右側）ペインにはテキストの直接入力またはMarkdownファイルのアップロードが可能です。 |

### その他

### コードブロック

`Ctrl+Alt+C` またはツールバーでコードブロックを挿入できます。

- TypeScript、JavaScript、Python、Java、Go、Rust、C/C++、SQL、Bash、HTML、CSS など約37言語のシンタックスハイライトに対応しています。
- 言語は自動検出されますが、ツールバーの言語ラベルから手動で指定することもできます。
- ツールバーから折りたたみ/展開、全画面編集、コードコピー、削除の操作が可能です。

### アドモニション

アドモニション（コールアウトブロック）は、色付きの枠と背景で重要な情報を強調表示します。

- スラッシュコマンド（`/note`, `/tip`, `/important`, `/warning`, `/caution`）で挿入できます。
- ソースモードでは、GitHub互換の記法を使用できます：`> [!NOTE]`, `> [!TIP]`, `> [!IMPORTANT]`, `> [!WARNING]`, `> [!CAUTION]`。

| タイプ | 色 | 説明 |
| ------ | -- | ---- |
| Note | 青 | 一般的な情報や参考事項。 |
| Tip | 緑 | 役立つアドバイスやベストプラクティス。 |
| Important | 紫 | 見逃してはいけない重要な情報。 |
| Warning | 黄 | 注意すべき潜在的な問題。 |
| Caution | 赤 | 危険な操作に関する重大な警告。 |

### 脚注

ドキュメントに脚注参照を挿入できます。

- `/footnote` スラッシュコマンドで脚注参照を挿入できます。
- ソースモードでは、`[^id]` で参照、`[^id]: 内容` で脚注定義を記述します。

### 数式（KaTeX）

KaTeX記法を使って数式を記述できます。

- `/math` スラッシュコマンドで数式ブロックを挿入できます。
- インライン数式は `$...$` で囲みます（例: `$E = mc^2$`）。

| ![全画面](/help/icons/fullscreen.png) | 全画面 | 数式を全画面モードで編集します。 |
| ![サンプル](/help/icons/sample.png) | サンプル挿入 | 定義済みの数式サンプル（分数、総和、積分、行列、極限、偏微分、場合分け）を挿入します。 |
| ![コピー](/help/icons/copy.png) | コードコピー | KaTeXソースコードをクリップボードにコピーします。 |

### 目次

ドキュメント内の見出しから目次を自動生成します。

- `/toc` スラッシュコマンドまたはツールバーのTOCボタンで挿入できます。
- ドキュメント内のすべての見出し（H1〜H6）からMarkdownリンクリストが生成されます。

### 画像

さまざまな方法で画像の挿入やMarkdownファイルのインポートができます。

- ツールバーボタンからURLを指定して画像を挿入できます。
- 画像ファイルをエディタに直接ドラッグ＆ドロップできます。
- クリップボードから画像を貼り付けできます（Ctrl+V / Cmd+V）。
- .mdファイルをエディタにドラッグ＆ドロップしてコンテンツをインポートできます。

## 比較・マージ

2つのドキュメントを並べて比較・マージできます。

- ツールバー右側の通常モード/比較モードのボタングループで切り替えます。
- 比較モードではエディタが左右2ペインに分割され、差分がハイライト表示されます。
- セカンダリ（右側）ペインには、テキストの直接入力またはMarkdownファイルのアップロードが可能です。
- 差分の取り込み（マージ）操作ができます。

## アウトライン

ドキュメント構造を表示し、セクション間をナビゲートできます。

![アウトラインパネル](/help/outline.png)

| アイコン | 操作 | 説明 |
| :------: | ---- | ---- |
| ![アウトライン](/help/icons/outline.png) | アウトライン表示 | ツールバーからアウトラインパネルの表示/非表示を切り替えます。 |

- アウトライン内の見出しをクリックすると、エディタ内のその位置にジャンプします。
- セクションの折りたたみ/展開で、見出し下のコンテンツを折りたたんだり展開できます。
- ドラッグ＆ドロップまたは上下移動ボタンでセクションを並べ替えできます。
- パネル境界をドラッグしてアウトラインパネルの幅を調整できます。

## コメント

ドキュメントにコメントや注釈を追加できます。

- テキストを選択して Ctrl+Shift+M を押すか、`/comment` スラッシュコマンドでコメントを追加できます。
- 選択テキストへのコメントは黄色の背景でハイライト表示されます。
- テキスト未選択時のコメントはカーソル位置にマーカーとして挿入されます。
- ツールバーからコメントパネルを開いて、コメントの確認・解決・管理ができます。
- フィルター機能：すべて・未解決・解決済みで絞り込みできます。

## エディタ設定

エディタの外観と動作をカスタマイズできます。

![エディタ設定](/help/settings.png)

- 調整可能な設定：行間、フォントサイズ、テーブル幅、エディタ最小幅、エディタ背景色。
- 見出し番号：「見出し番号」スイッチで、見出しの自動採番（例: 1, 1.1, 1.1.1）を切り替えできます。
- 改ページガイド：設定パネルの「改ページガイド」スイッチで、エディタ内の改ページ位置の表示/非表示を切り替えられます。

### エンコーディング・改行コード

エディタ下部のステータスバーから、ファイルのエンコーディングや改行コードを変更できます。

- ステータスバーのエンコーディング表示（例: "UTF-8"）をクリックすると、UTF-8 / Shift_JIS / EUC-JP から選択できます。変換前に確認ダイアログが表示されます。
- ステータスバーの改行コード表示（"LF" または "CRLF"）をクリックすると、LFとCRLFを切り替えられます。
