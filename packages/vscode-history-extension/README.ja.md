リポジトリ管理・変更のステージング・コミットグラフ・ファイル履歴の閲覧を、VS Code のサイドバーから一元操作できます。


## リポジトリ

フォルダーを開くかリポジトリをクローンして、ドラッグ&ドロップ対応のファイルツリーでプロジェクトを操作します。

- **フォルダーを開く** — ローカルディレクトリをリポジトリルートとして開く
- **リポジトリをクローン** — リモートリポジトリをローカルにクローン
- **ブランチ切替** — ツリーから直接ブランチを切り替え
- **ファイル操作** — 新規ファイル・フォルダー、名前変更、削除、インポート、切り取り・コピー・貼り付け
- **Markdown フィルター** — `.md` ファイルのみを表示するフィルターを切替


## 変更

変更のステージングとコミットをサイドバーのインライン操作で完結させます。

- **ステージ済み** / **変更**（未ステージ）のファイルをグループ表示
- サイドバーバッジで変更ファイル数を常時表示
- ファイル単位またはまとめて **ステージ / アンステージ**
- 未ステージの変更をファイル単位またはまとめて **破棄**
- 変更パネルのメニューから **コミット** と **プッシュ**


## グラフ

ブランチの履歴を ASCII コミットグラフで可視化します。

- オンデマンドでグラフを更新
- コミットメッセージと並べてブランチの流れを表示


## タイムライン

ファイルごとのコミット履歴を一覧して、リビジョン間の差分を比較します。

- ファイルごとのコミット一覧（日付順）
- **このコミットと比較** — 任意の過去リビジョンをサイドバイサイドの差分で表示
