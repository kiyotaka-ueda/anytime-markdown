## [0.8.0] - 2026-03-25

### 追加
- 11種類のノード: 矩形、楕円、ひし形、平行四辺形、円柱、付箋、テキスト、インサイト、ドキュメント、フレーム、画像
- 3種類のエッジ: 直線、矢印、コネクタ
- A* 障害物回避ルーティングによる直交コネクタ
- ベジェ曲線エッジ（手動制御ポイント対応）
- 全ノード型・グラデーション対応の SVG エクスポート
- draw.io XML エクスポート/インポート（Node.js 用 xmldom 互換）
- スナップ整列付きスマートガイド
- ノードとリサイズのグリッドスナップ
- ノード整列（左、右、上、下、中央）と分配
- ビューポート: パン、ズーム（0.1〜10倍）、コンテンツフィット
- easeOutCubic イージングによるアニメーションズーム
- レンダリング最適化のためのビューポートカリング
- フレームノード（グループ化用）
- ノードのロック機能と z-index レイヤー管理
- ドラッグ＆ドロップ画像配置
- ノード URL ハイパーリンク
- エッジラベルとエンドポイント形状設定（矢印、円、ひし形、バー）
- 追加接続ポイントのカスタマイズ
- 選択状態を保持した Undo/Redo（履歴上限 50）
- MCP サーバー: `mcp-graph`（12ツール: CRUD、エクスポート/インポート、ノード/エッジ操作）

### 変更
- wrapText を文字単位から単語単位計測に最適化
- A* のオープンセットをバイナリヒープに置換（O(log n)）
- レンダリングループをシングルパスノードパーティショニングに最適化
- drawNode の if-else チェーンをシェイプレジストリパターンに置換
- z-order 管理を zIndex プロパティに統一
- 接続ポイント計算を getConnectionPoints に統一
- 共有定数を constants.ts に集約

### 修正
- テキストオーバーフロー時の省略表示（クリッピング）
- キャンバス外での mouseup によるドラッグ固着の防止
- ロックされたノードの削除を防止
- テキスト入力中のツールショートカット誤発火を防止
- コントラスト、テキスト色の一貫性、選択表示の改善
- importFromDrawio での不正 XML 検出
- エッジラベル背景テーマの一貫性

### アクセシビリティ
- Canvas の ARIA ロール、ラベル、ライブステータス領域
- 矢印キーによるノード移動のキーボードナビゲーション
- カラーパレットの ARIA ロール付きキーボードナビゲーション
- PropertyPanel コントロールのラベル
- prefers-reduced-motion 対応
- ノード追加/削除の aria-live アナウンス
